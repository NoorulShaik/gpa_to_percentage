# 🎓 GPA to Percentage Converter

A lightweight, lightning-fast web application built with **React** and **Vite** that allows students to instantly convert their GPA/CGPA into a percentage. 

🚀 **Live Demo:** [Check out the live app here!](https://noorulshaik.github.io/gpa_to_percentage/)

---

## 🛠️ Tech Stack & Concepts Learned

This project was built to master the fundamentals of modern frontend development, build tools, and deployment pipelines:

* **React.js:** Implemented dynamic state management using hooks (`useState`) for real-time calculations.
* **Vite:** Leveraged as a modern, ultra-fast build tool and development server.
* **CSS3:** Custom styled for a clean, intuitive, and fully responsive user interface.
* **CI/CD Deployment:** Configured automated deployment pipelines using the `gh-pages` package to seamlessly host production builds on GitHub Pages.

---

## 🚀 Features

- **Real-Time Conversion:** Percentage updates instantly as the user types their GPA.
- **Form Validation:** Prevents invalid inputs (e.g., negative numbers or values exceeding the grading scale).
- **Responsive Design:** Optimized for seamless viewing on mobile phones, tablets, and desktops.

---

## 🔮 Future Features Pipeline

This project is a continuous learning endeavor. Upcoming enhancements include:
1. **Dynamic Formula Selection:** Support for multiple university standards (e.g., AICTE, Mumbai University, 10-point scale vs. 4-point scale).
2. **Multi-Semester Tracker:** Dynamically add semesters to calculate cumulative overall CGPA.
3. **Local Storage Integration:** Persist calculation history locally so users don't lose data on page refresh.
4. **Dark Mode toggle:** Enhanced UI customization for better accessibility.

## 💻 Local Development Setup

To run this project locally on your machine, follow these steps:

    1. **Clone the repository:**
       ```bash
       git clone [https://github.com/noorulshaik/gpa_to_percentage.git](https://github.com/noorulshaik/gpa_to_percentage.git)
       cd gpa_to_percentage
       
    2. Install dependencies:
    npm install
    
    3.Start the development server:
    npm run dev
    Open http://localhost:5173 in your browser to view the application.
    
    4.Build and Deploy:
    npm run deploy
